# DoubtSync – Smart YouTube Doubt Tracker

## 🚀 Overview
DoubtSync is a Chrome extension that helps students capture and revisit doubts while watching YouTube videos.

## 🎯 Problem
Students often forget their doubts while watching long tutorials and cannot revisit the exact moment later.

## 💡 Solution
DoubtSync allows users to save doubts along with timestamps and revisit them instantly.

## ✨ Features
- Save doubts with timestamp
- Jump to exact video moment
- Delete/manage doubts
- Clean and simple UI

## ⚙️ Installation
1. Download the project
2. Open Chrome → chrome://extensions/
3. Enable Developer Mode
4. Click Load Unpacked
5. Select the project folder

## 📌 Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extension APIs
